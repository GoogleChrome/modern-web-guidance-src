# Google Chrome Agent Skills

A curated collection of web development agent skills and tools.

## Installation

### For Claude Code
```bash
/plugin marketplace add GoogleChrome/skills-alpha
/plugin install googlechrome-skills@skills-alpha
/reload-plugins
```

### For Gemini CLI
```bash
gemini extensions install https://github.com/GoogleChrome/skills-alpha --auto-update
```

### For Vanilla Skills Pack
```bash
npx skills add GoogleChrome/skills-alpha
```
