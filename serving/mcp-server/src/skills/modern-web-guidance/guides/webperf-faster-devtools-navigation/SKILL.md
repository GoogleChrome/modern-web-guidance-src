# Faster DevTools Navigation

## Best Practices

This guide focuses on enhancing your DevTools workflow by leveraging keyboard shortcuts and customizable settings for faster navigation and panel management.

### Use the Drawer

The **Drawer** is a collapsible panel at the bottom of DevTools that allows you to quickly access frequently used panels and tools.

*   **Open/Close Drawer:** Press <kbd>Escape</kbd> to toggle the Drawer.
*   **Access More Tools:** Click the <img src="image/more-tools-b0cd3b22a455d.svg" alt="More Tools." width="24" height="24"> **More Tools** icon within the Drawer to open additional tabs like the **Command Menu** or **Sensors**.
*   **Customize Drawer Content:** Reorder panels and Drawer tabs by dragging them up or down. You can also move panels from the main DevTools window into the Drawer and vice-versa to create a personalized layout.

### Leverage the Command Menu

The **Command Menu** provides a unified interface for executing commands, toggling settings, and opening panels and tabs, even if you don't remember their exact location.

*   **Open Command Menu:** Press <kbd>Control+Shift+P</kbd> (Windows/Linux) or <kbd>Command+Shift+P</kbd> (macOS).
*   **Search and Execute:** Type keywords to find and execute DevTools commands, such as "Show rendering" or "Capture full size screenshot."

### Optimize Layout and Shortcuts

Customize DevTools to suit your workflow by adjusting the layout and creating custom keyboard shortcuts.

*   **Layout Options:** In **Settings** > **Preferences**, you can change the default layout from `auto` to always `horizontal` or `vertical` for panels. This can help prevent content reflow and keep your workspace consistent.
*   **Customize Shortcuts:** Enable the **Keyboard shortcut editor** in **Settings** > **Experiments**. This allows you to rebind existing shortcuts or create new ones for DevTools commands and actions, further personalizing your workflow.

For a more hands-on learning experience, see the [Customize DevTools](/docs/devtools/customize) tutorial.
