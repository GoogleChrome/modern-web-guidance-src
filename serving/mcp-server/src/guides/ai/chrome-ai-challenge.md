## Overview

The Google Chrome Built-in AI Challenge 2025 invites developers to build and submit new web applications or Chrome Extensions that leverage one or more built-in AI APIs. This includes powerful models like Gemini Nano, offering a chance to win prizes and connect with the Chrome AI team.

## Key APIs and Features

This year's challenge includes enhanced and new APIs:

*   **Prompt API**: Now supports multimodal input (image and audio).
*   **Proofreader API**: A new addition for text correction.
*   **Summarizer API**: Condense longer texts into concise summaries.
*   **Translator API**: Translate content between languages.
*   **Language Detector**: Identify the language of a given text.
*   **Writer API**: Assist in content generation.
*   **Rewriter API**: Help rephrase or improve existing text.

## Hybrid AI Solutions

For broader reach, including mobile users, consider implementing a hybrid AI solution by integrating with:

*   **Firebase AI Logic**: For managing and deploying AI models.
*   **Gemini Developer API**: To access advanced AI capabilities.

## Submission and Judging Criteria

Submissions will be evaluated based on:

*   Functionality
*   Purpose
*   Content
*   User Experience
*   Technical Execution

## Prizes and Timeline

*   **Prizes**: 16 prizes ranging from $200 to $14,000 USD, totaling $70,000 USD.
*   **Challenge Period**: Started already and runs until 11:45 PM PST on October 31, 2025.
