# Scroll Elements Into View

This guide demonstrates how to make elements visible in the viewport when they are not currently in view, using browser developer tools.

## Best Practices

When inspecting an element in the Elements panel and it's not visible on the page, you can use the "Scroll into view" option to bring it into the viewport.

**Steps:**

1.  **Right-click** on the DOM node you wish to inspect in the Elements panel.
2.  Select **"Scroll into view"** from the context menu.

This action will automatically adjust the page's scroll position to make the selected element visible.

Additionally, remember that hovering over a DOM node in the Elements panel provides a tooltip displaying its dimensions and position, which can be helpful for understanding its layout and visibility.
