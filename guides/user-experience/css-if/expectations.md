# Expectations

- The `.card` element MUST have a `background-color` of `rgb(255, 255, 255)` and `color` of `rgb(0, 0, 0)` by default.
- When `document.body` has the class `dark`, the `.card` element MUST have a `background-color` of `rgb(0, 0, 0)` and `color` of `rgb(255, 255, 255)`.
- When the viewport width is strictly less than `600px` (e.g. `500px`), the `.card` element MUST have a `padding` of `16px`.
- When the viewport width is `600px` or greater (e.g. `800px`), the `.card` element MUST have a `padding` of `32px`.
