Hover to expand text

This is the first line of text. When you hover over this box, the container will smoothly animate from a fixed height of one line (1lh) to height: auto using the CSS calc-size() function. This was previously impossible with pure CSS without using max-height workarounds. By using calc-size(auto, size), we tell the browser to calculate the intrinsic size and allow it to be used in a transition. This allows for fluid, content-aware UI components like accordions and menus that don't need fixed pixel values or JavaScript to animate perfectly.

Uses: height: calc-size(auto, size);
