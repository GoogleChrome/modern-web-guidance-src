Colored Scrollbar

This box demonstrates a scrollable container with customized scrollbar colors.

In browsers that support the standard properties, the scrollbar-color: #00bcd4 #e0f7fa property is used.

In browsers that only support legacy vendor prefixes, ::-webkit-scrollbar-thumb and ::-webkit-scrollbar-track are used to color the scrollbar.
