Browser Features for Canvas Content

This demo shows how HTML-in-Canvas enables integration with browser features like Translation or Autofill.

This text is rendered inside the canvas but is present in the DOM tree. Browser features like translation services and reading mode can access this content because it is part of the layout subtree.

It even supports Autofill! If you have an email address saved in your browser, try to focus on the input field and trigger the Autofill dropdown.
