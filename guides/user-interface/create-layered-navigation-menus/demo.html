Swipeable Navigation Demo

This demo uses a scroll-snap overlay technique with proper ::backdrop styling.

Key Features:

  • ::backdrop: Uses the native backdrop pseudo-element for dimming.
  • Swipe to Close: Drag the menu to the left to dismiss it.
  • Scroll Snap: The menu snaps naturally to open or closed states.
  • Top Layer: Uses popover="manual" for z-index management.

Try opening the menu and swiping it closed!
